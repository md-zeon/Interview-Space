# How to align text in HTML?

## Overview

Text alignment in HTML can be achieved through multiple methods. While HTML provides some alignment attributes, CSS is the modern and recommended approach for controlling text alignment and layout.

## Methods of Text Alignment

### 1. HTML Align Attribute (Deprecated but Supported)

#### Using align Attribute

Some HTML elements support the `align` attribute for basic text alignment:

```html
<!-- Center align heading (deprecated usage) -->
<h1 align="center">Centered Heading</h1>

<!-- Right align paragraph (deprecated usage) -->
<p align="right">Right-aligned paragraph text.</p>

<!-- Justified text (deprecated usage) -->
<p align="justify">
	This text will be justified, spreading evenly across the width of the
	container.
</p>
```

#### Supported Elements for align Attribute

- `div`, `p`, `h1-h6` - Block elements
- `td`, `th` - Table cells
- `caption` - Table captions

#### Limitations

- Deprecated in HTML5 (use CSS instead)
- Limited alignment options
- Not responsive
- Interferes with CSS cascading

### 2. CSS Text-Align Property (Recommended)

#### Basic Text Alignment

The `text-align` CSS property controls horizontal text alignment:

```html
<style>
	.center-aligned {
		text-align: center;
	}

	.right-aligned {
		text-align: right;
	}

	.left-aligned {
		text-align: left;
	}

	.justified {
		text-align: justify;
	}
</style>
```

```html
<div class="center-aligned">
	<h2>Centered Content</h2>
	<p>This paragraph is centered.</p>
</div>

<div class="right-aligned">
	<h2>Right Aligned</h2>
	<p>This paragraph is right-aligned.</p>
</div>

<div class="left-aligned">
	<h2>Left Aligned (Default)</h2>
	<p>This paragraph is left-aligned.</p>
</div>

<div class="justified">
	<h2>Justified Text</h2>
	<p>
		This paragraph uses justified alignment, where text is spread evenly across
		the container width.
	</p>
</div>
```

#### Inline CSS Method

```html
<p style="text-align: center;">Centered paragraph using inline CSS</p>
```

#### External CSS Method

```css
/* styles.css */
.center-text {
	text-align: center;
}

.right-text {
	text-align: right;
}

.justify-text {
	text-align: justify;
}
```

```html
<!-- index.html -->
<link
	rel="stylesheet"
	href="styles.css" />

<p class="center-text">Centered with external CSS</p>
<p class="right-text">Right aligned with external CSS</p>
<p class="justify-text">Justified with external CSS</p>
```

## Advanced Text Alignment Techniques

### 3. CSS Flexbox for Centering

#### Centering Text Vertically and Horizontally

```css
.flex-center {
	display: flex;
	justify-content: center; /* Horizontal centering */
	align-items: center; /* Vertical centering */
	height: 200px; /* Fixed height for vertical centering */
}
```

```html
<div class="flex-center">
	<p>This text is centered both horizontally and vertically.</p>
</div>
```

### 4. CSS Grid for Advanced Layout

```css
.grid-center {
	display: grid;
	place-items: center; /* Centers both horizontally and vertically */
	height: 200px;
}
```

```html
<div class="grid-center">
	<div>
		<h3>Centered Title</h3>
		<p>This content is centered using CSS Grid.</p>
	</div>
</div>
```

### 5. Using Auto Margins

```css
.auto-center {
	width: 50%; /* Fixed width */
	margin: 0 auto; /* Auto left/right margins center horizontally */
}
```

```html
<div class="auto-center">
	<h2>Centered Block</h2>
	<p>This block-level element is centered horizontally.</p>
</div>
```

## Text Alignment for Specific Elements

### Table Cell Alignment

```html
<table>
	<tr>
		<th style="text-align: left;">Left Aligned Header</th>
		<th style="text-align: center;">Center Aligned Header</th>
		<th style="text-align: right;">Right Aligned Header</th>
	</tr>
	<tr>
		<td style="text-align: left;">Left data</td>
		<td style="text-align: center;">Center data</td>
		<td style="text-align: right;">Right data</td>
	</tr>
</table>
```

### Form Element Alignment

```html
<form style="text-align: center;">
	<label for="name">Name:</label>
	<input
		type="text"
		id="name"
		required />
	<br />
	<button type="submit">Submit</button>
</form>
```

### List Alignment

```html
<ul style="text-align: center;">
	<li>Centered list item</li>
	<li>Another centered item</li>
</ul>
```

## Responsive Text Alignment

### Media Query Based Alignment

```css
.responsive-text {
	text-align: left; /* Default for small screens */
}

@media (min-width: 768px) {
	.responsive-text {
		text-align: center; /* Center on larger screens */
	}
}

@media (min-width: 1200px) {
	.responsive-text {
		text-align: right; /* Right align on very large screens */
	}
}
```

```html
<div class="responsive-text">
	<p>This text changes alignment based on screen size.</p>
</div>
```

## Vertical Text Alignment

### Using Line-Height

```css
.tall-box {
	height: 100px;
	line-height: 100px; /* Vertically centers single line */
}
```

### Using Flexbox

```html
<style>
	.vertical-center {
		display: flex;
		align-items: center; /* Vertical centering */
		height: 150px;
	}
</style>

<div class="vertical-center">
	<p>This single line of text is vertically centered.</p>
</div>
```

### Multiline Vertical Centering

```css
.multi-line-center {
	display: flex;
	align-items: center;
	justify-content: center;
	flex-direction: column;
	height: 200px;
	text-align: center;
}
```

## Best Practices

### 1. Prefer CSS over HTML Attributes

```html
<!-- Avoid (deprecated) -->
<p align="center">Centered text</p>

<!-- Prefer (CSS) -->
<p style="text-align: center;">Centered text</p>
```

### 2. Use Semantic HTML Structure

```html
<!-- Good structure -->
<header style="text-align: center;">
	<h1>Site Title</h1>
	<p>Tagline here</p>
</header>

<main>
	<article style="text-align: left;">
		<!-- Default left alignment -->
		<h2>Article Title</h2>
		<p>Content...</p>
	</article>
</main>
```

### 3. Consider Mobile Responsiveness

```css
/* Center on large screens, left-align on mobile */
.centered-content {
	text-align: left;
}

@media (min-width: 768px) {
	.centered-content {
		text-align: center;
	}
}
```

### 4. Maintain Readability

- Avoid justified text for web (works better in print)
- Ensure sufficient contrast
- Consider line length (50-75 characters optimal)

## Troubleshooting Common Issues

### Issue: Text-Align Not Working

**Possible causes:**

- Element needs to be block-level
- More specific CSS rule overriding
- Inline elements don't respond to text-align

```html
<!-- This won't work (span is inline) -->
<span style="text-align: center;">Not centered</span>

<!-- This works -->
<div style="text-align: center;">
	<span>Now centered</span>
</div>
```

### Issue: Vertical Centering Problems

**Solutions:**

```css
/* For single line */
.single-line {
	line-height: 50px;
	height: 50px;
}

/* For multi-line with flexbox */
.multi-line {
	display: flex;
	align-items: center;
	min-height: 100px;
}
```

### Issue: Center Not Truly Centered

**Check for:**

- Parent container width constraints
- Box-sizing property
- Padding or margins affecting layout

## Summary

- **HTML align attribute**: Deprecated but works for basic alignment
- **CSS text-align**: Primary method for horizontal alignment (`left`, `center`, `right`, `justify`)
- **Flexbox**: Best for centering both horizontally and vertically
- **CSS Grid**: Modern layout with centering capabilities
- **Auto margins**: Simple horizontal centering for fixed-width elements

Modern web development favors CSS-based solutions for text alignment, providing better control, responsiveness, and maintainability.
