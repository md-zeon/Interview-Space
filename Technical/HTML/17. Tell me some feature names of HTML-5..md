# Tell me some feature names of HTML5

HTML5 brought revolutionary changes to web development with dozens of new features and improvements. Below is a comprehensive list of major HTML5 features, categorized for easier understanding.

## Semantic Elements

### Structural Elements

```html
<!-- Semantic page structure -->
<header>Site header with logos and navigation</header>
<nav>Primary navigation menu</nav>
<main>Main content of the page</main>
<article>Self-contained content like blog posts</article>
<section>Thematic grouping of content</section>
<aside>Sidebar or supplementary content</aside>
<footer>Page footer with copyright and links</footer>
```

- **`<header>`**: Introductory content, typically contains site logo and main navigation
- **`<nav>`**: Section with navigation links to same/other pages
- **`<main>`**: Main content (one per page)
- **`<article>`**: Independent, distributable content (blogs, news, comments)
- **`<section>`**: Thematic grouping of content
- **`<aside>`**: Indirectly related content (sidebars, related links)
- **`<footer>`**: Footer content (copyright, contact info, links)

### Other Semantic Elements

```html
<time datetime="2023-12-31">New Year's Eve</time>
<mark>Highlighted text</mark>
<summary>Details summary (click to expand)</summary>
<details>Expandable content section</details>
<address>Contact information</address>
<data value="12345">Product ID 12345</data>
```

- **`<time>`**: Date/time with machine-readable format
- **`<mark>`**: Highlighted or marked text
- **`<details>`/`<summary>`**: Disclosure widget (accordion)
- **`<address>`**: Contact information for person/organization
- **`<data>`**: Machine-readable value with human-readable equivalent

## Multimedia Elements

### Audio and Video

```html
<!-- Basic audio player -->
<audio
	controls
	src="music.mp3">
	<source
		src="music.ogg"
		type="audio/ogg" />
	Your browser doesn't support audio.
</audio>

<!-- Basic video player -->
<video
	controls
	width="640"
	height="360">
	<source
		src="movie.mp4"
		type="video/mp4" />
	<source
		src="movie.webm"
		type="video/webm" />
	Your browser doesn't support video.
</video>
```

- **`<audio>`**: Native audio playback without plugins
- **`<video>`**: Native video playback without plugins
- **`<source>`**: Multiple media sources for compatibility
- **`<track>`**: Subtitles, captions, and chapters

### Canvas and Graphics

```html
<!-- Drawing canvas -->
<canvas
	id="myCanvas"
	width="200"
	height="100">
	Canvas not supported.
</canvas>

<script>
	const canvas = document.getElementById("myCanvas");
	const ctx = canvas.getContext("2d");
	ctx.fillRect(10, 10, 50, 50); // Draw rectangle
</script>
```

- **`<canvas>`**: JavaScript-based drawing surface
- **`<svg>`**: Scalable vector graphics (XML-based)

## Form Enhancements

### New Input Types

```html
<!-- Various form input types -->
<input
	type="email"
	placeholder="email@example.com" />
<input
	type="url"
	placeholder="https://example.com" />
<input
	type="tel"
	placeholder="555-1234" />
<input
	type="number"
	min="1"
	max="100"
	step="1" />
<input
	type="range"
	min="0"
	max="100"
	value="50" />
<input type="date" />
<input type="time" />
<input type="datetime-local" />
<input type="color" />
<input type="search" />
```

- **`email`**: Email validation
- **`url`**: URL format validation
- **`tel`**: Telephone number input
- **`number`**: Numeric input with spinners
- **`range`**: Slider control
- **`date`/`time`**: Date/time pickers
- **`datetime-local`**: Local date/time
- **`color`**: Color picker
- **`search`**: Search field

### Form Attributes

```html
<form
	action="/submit"
	method="post">
	<input
		type="text"
		required
		autofocus
		placeholder="Enter name" />
	<input
		type="password"
		autocomplete="current-password" />
	<textarea
		minlength="10"
		maxlength="500">
Initial text</textarea
	>
	<button
		type="submit"
		formnovalidate>
		Submit without validation
	</button>
</form>
```

- **`required`**: Field must be filled
- **`autofocus`**: Field gets focus on page load
- **`placeholder`**: Hint text
- **`autocomplete`**: Browser autofill hints
- **`minlength`/`maxlength`**: Text length constraints
- **`formnovalidate`**: Skip validation for this submit button

## APIs and Programming Interfaces

### Geolocation API

```javascript
navigator.geolocation.getCurrentPosition(function (position) {
	console.log(position.coords.latitude, position.coords.longitude);
});
```

- **Purpose**: Get user's geographic location

### Drag and Drop API

```html
<div dragged="dragstart">Draggable element</div>
<div id="dropzone">Drop zone</div>

<script>
	const draggable = document.querySelector("[dragged]");
	const dropzone = document.getElementById("dropzone");

	draggable.draggable = true;
	dropzone.addEventListener("drop", handleDrop);
</script>
```

### Web Storage API

```javascript
// Local storage (persists)
localStorage.setItem("username", "john");

// Session storage (clears when tab closes)
sessionStorage.setItem("tempData", "temporary");

// Retrieve data
const user = localStorage.getItem("username");
```

- **`localStorage`**: Persistent client-side storage
- **`sessionStorage`**: Session-only storage

### Web Workers API

```javascript
// worker.js
self.addEventListener("message", function (e) {
	const result = e.data + 10;
	self.postMessage(result);
});

// main.js
const worker = new Worker("worker.js");
worker.postMessage(5);
worker.onmessage = function (e) {
	console.log(e.data); // 15
};
```

- **Purpose**: Run JavaScript in background threads

### WebSocket API

```javascript
const socket = new WebSocket("ws://example.com/socket");
socket.onopen = function () {
	socket.send("Hello Server!");
};
socket.onmessage = function (event) {
	console.log(event.data);
};
```

- **Purpose**: Real-time bidirectional communication

### History API

```javascript
// Add entry without page reload
history.pushState({ page: "about" }, "About", "/about");

// Listen for back button
window.addEventListener("popstate", function (event) {
	console.log("Location changed:", event.state);
});
```

- **Purpose**: Manipulate browser history

## Offline and Storage

### Application Cache (deprecated, replaced by Service Workers)

```html
<!DOCTYPE html>
<html manifest="app.cache">
	<!-- App content -->
</html>

<!-- app.cache -->
CACHE MANIFEST index.html styles.css app.js NETWORK: * FALLBACK: / /offline.html
```

### Service Workers API

```javascript
// Register service worker
navigator.serviceWorker.register("/sw.js");

// In sw.js
self.addEventListener("install", function (event) {
	event.waitUntil(
		caches.open("my-cache").then(function (cache) {
			return cache.addAll(["/index.html", "/styles.css"]);
		}),
	);
});
```

- **Purpose**: Progressive web app capabilities, offline functionality

## Communication APIs

### PostMessage API

```javascript
// Send message to iframe
const iframe = document.querySelector("iframe");
iframe.contentWindow.postMessage("Hello from parent", "*");

// Receive message in iframe
window.addEventListener("message", function (event) {
	if (event.origin !== "https://trusted-site.com") return;
	console.log(event.data);
});
```

### BroadcastChannel API

```javascript
const channel = new BroadcastChannel("chat");
channel.postMessage("Hello everyone!");
channel.onmessage = function (event) {
	console.log("Received:", event.data);
};
```

## Performance Features

### Lazy Loading

```html
<!-- Lazy load images -->
<img
	src="image.jpg"
	loading="lazy"
	alt="Lazy loaded image" />

<!-- Lazy load iframes -->
<iframe
	src="video.html"
	loading="lazy"></iframe>
```

### Preconnect

```html
<!-- Hint browser to preconnect -->
<link
	rel="preconnect"
	href="https://fonts.googleapis.com" />
<link
	rel="preconnect"
	href="https://fonts.gstatic.com"
	crossorigin />

<link
	href="https://fonts.googleapis.com/css2?family=Roboto&display=swap"
	rel="stylesheet" />
```

### Prefetch

```html
<!-- DNS prefetch -->
<link
	rel="dns-prefetch"
	href="//cdn.example.com" />

<!-- Resource hints -->
<link
	rel="prefetch"
	href="/next-page.html" />
<link
	rel="preload"
	href="critical.css"
	as="style" />
```

## Security Features

### Content Security Policy

```html
<meta
	http-equiv="Content-Security-Policy"
	content="default-src 'self'; script-src 'self' https://trusted.cdn.com" />
```

### Referrer Policy

```html
<!-- Control how much referrer info is sent -->
<meta
	name="referrer"
	content="strict-origin-when-cross-origin" />
```

### X-Frame-Options (deprecated, use CSP)

```html
<meta
	http-equiv="X-Frame-Options"
	content="DENY" />
```

## Summary of Major HTML5 Features

### **Semantic Web**

- Structural elements (`<header>`, `<nav>`, `<main>`, etc.)
- Improved document outline
- Better accessibility

### **Rich Media**

- Native audio/video playback
- Canvas for dynamic graphics
- SVG support

### **User Interaction**

- Advanced form controls
- Drag and drop
- Geolocation

### **Offline Capabilities**

- Service workers
- Web storage
- Application cache

### **Real-time Communication**

- WebSocket API
- Server-sent events
- PostMessage API

### **Performance**

- Lazy loading
- Resource hints
- Web workers

### **Security**

- Content Security Policy
- Referrer Policy
- CORS improvements

HTML5 transformed web development from static documents to interactive, app-like experiences while maintaining backward compatibility and improving accessibility standards.
